# 🌍 Air Pollution Analysis & Forecasting System

A full-stack data analytics and machine learning platform that visualizes global air quality trends, provides real-time AQI monitoring, and predicts future air quality using advanced ensemble machine learning models.

---

## 📌 Project Overview

Air pollution affects millions of people worldwide and remains one of the most critical environmental challenges. **Pathways to Purity** is a comprehensive web application that combines:

- 🌎 Real-time Air Quality Monitoring
- 🗺️ Interactive Geospatial Visualization
- 📊 Historical AQI Analytics
- 🤖 Machine Learning Forecasting
- 🔬 Pollutant Correlation Analysis
- 🏥 AQI Health Advisory System

The platform tracks air quality across **40+ major cities worldwide**, enabling users, researchers, and policymakers to understand pollution patterns and make data-driven decisions.

---

## 🎯 Key Features

### 🌎 Global AQI Dashboard

- Interactive world map displaying city-wise AQI levels
- Color-coded pollution indicators
- Real-time AQI monitoring
- User location detection
- Top 5 cleanest cities
- Top 5 most polluted cities

### 📊 Historical Air Quality Analytics

- AQI Trend Analysis
- Pollutant Distribution Analysis
- Monthly & Yearly AQI Trends
- Weather Correlation Analysis
- AQI Category Distribution
- Seasonal Pollution Insights

### 🔬 Advanced Data Visualization

- Interactive Line Charts
- Scatter Plots
- Radar Charts
- Bar Charts
- Heatmaps
- Geospatial Mapping

### 🧮 AQI Calculator

- EPA-standard AQI calculation
- PM2.5-based AQI estimation
- Health category classification

### 🤖 Machine Learning Forecasting

- Forecast AQI up to 90 days ahead
- City-specific predictive models
- Confidence Interval Estimation
- Health Risk Predictions
- Automated Model Selection

### 🌤 Live AQI Widgets

- 80+ global city widgets
- Weather integration
- Auto-refresh mechanism
- AQI gauges and pollutant breakdowns

### Architecture Overview

- **Frontend:** React.js
- **Backend API:** Node.js + Express
- **Database:** MySQL
- **ML Service:** Python Flask
- **Visualization:** Chart.js, Recharts
- **Mapping:** MapLibre GL, Leaflet
- **Machine Learning:** Scikit-Learn, XGBoost, LightGBM
